# elk
An educational, roleplay game for teachers to practice eliciting learner knowledge

Setup
---
 
```
npm install
```
 
 
Compile
---
 
```
npm run compile 
```

Run
---
 
```
node server.js
```

Deployed at https://tsl-elk.herokuapp.com/
